Ты реализуешь фичу `$1` для AI Trading Agent.

## Шаги

1. Прочитай `CLAUDE.md` — архитектура проекта.
2. Прочитай `docs/$1.md` (ТЗ) и `docs/plan/$1.md` (план) если есть.
3. Открой `docs/tasklist/$1.md`, найди первую задачу с `- [ ]`.
4. Сформулируй план изменений:
    - какие файлы затронуты
    - какие методы добавляются/меняются
5. Остановись и спроси подтверждение пользователя.
6. После подтверждения:
    - внеси минимальные изменения для текущей задачи
    - отметь задачу как `- [x]` в tasklist
    - покажи diff и кратко опиши что сделано
7. Не переходи к следующей задаче без явной команды.

## Правила
- Следуй паттернам проекта (NestJS модули, Prisma, styled-components на фронте).
- Новые модули подключай в app.module.ts.
- TypeScript строго, без any без причины.
- После изменения Prisma схемы — напомни про `npx prisma generate`.
- Не трогай файлы вне текущей задачи.
